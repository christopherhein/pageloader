Alright finish it up. All you need to do it link the two Javascript files, by placing this in the <head>:
<script src="js/jquery.min-1.4.2.js"></script>
<script src="js/jquery.pageloader.min-0.0.2.js"></script>

Now for your content code in the index.html page, you need to place this where ever the content files that you created on files page should be loaded into: <div id="content-container">
<div class="content">

</div>
</div>

Now that the files are linked correctly and the content divs are in place you can than code out your pages for the first initial page, something like home.html, call the jQuery function (load_page) this will load up the content as your initial page loads with no transition. it should look like this: $('body').load_page({
page: 'overview.html',
parent: 'content-container',
child: 'content',
title: 'jQuery Page Loader || Overview'
});
Now as you can see your page link should be sent in the page parameter, and than set a title for what the title tag should read*

* to use this feature, add "id='pg_title'" to the title tag in the head of the document

So thats all working now, and now you want to make it even better with the transitions on every page load well, your in luck. each one of you navigation items now needs a jQuery function that corresponds to it's request, this will like very similar to the last function but wrapped in a click function like this: $('.required').click(function() {
$(this).load_newpage({
page: 'required.html',
parent: 'content-container',
child: 'content',
title: 'jQuery Page Loader || Required',
height: 100
});
});

just make sure you adjust all the heights if you pages have different page heights.
